# coding-pal

instructions, skills and agents to improve your AI coding assistant.

## Install

To install these instructions in an APM-enabled project, run:

```bash
apm install ALTEN-group/coding-pal/<type>/<name>
```

example : 

```bash
apm install ALTEN-group/coding-pal/instructions/sharp-agent
```

This will install the `instructions/sharp-agent.instructions.md` file into your repository. so once it is installed your agent can load it directly.
